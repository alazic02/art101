Lab 10 - JavaScript Events and Forms

Challenges

I didn't face many challenges while completing this lab (other than my crippling procrastination...) as I was able to use the class slides and work from Lab 9 as references on how to properly attach the event listener to the button and make the results of the sorting function appear in the 'output' div.

Problems

There weren't many problems that we encountered, aside from a few errors that came up when we were testing the button's functions. We ended up placing all of the active parts of the lab (sorting the input, making a new 'p' element, and appending it to the 'output' div) inside of the button's eventListener function. If we didn't do that, it would give us errors because variables inside of that function couldn't be accessed outside of it since they were not global variables. I assume a more efficient way to do it would be to have some global variables, but since the input and sorted name didn't need to be accessed anywhere else on the page, it didn't matter much for the purpose of this lab.
